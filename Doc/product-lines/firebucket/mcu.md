# 消防吊桶 MCU 职责

## 状态

- 状态：active
- 产品代码：`firebucket`

## 设备侧必须负责

- 上报 Link 在线 / 离线状态。
- 上报桶设备状态。
- 执行桶开关控制。
- 返回控制 ACK。
- 断线或异常时给 App 可判断状态。

## Link / 桶关系

- 一个账号可有多个 Link。
- 一个 Link 下可挂多个桶。
- App 展示和控制时必须避免串到其他 Link 或其他产品线。

## 安全约束

- 离线桶不可控制。
- 控制失败必须返回可诊断 ACK 或错误状态。
- 如果硬件存在互锁或安全保护，MCU 侧优先执行，App 只展示结果。

## 联调清单

- 登录后能看到消防吊桶设备。
- Link 下子桶数量和状态正确。
- App 控制单个桶后，MCU 返回对应 ACK。
- 悬浮窗选中设备后只控制当前设备。
